# Typing.dev

A terminal-style code typing trainer built for developers. Practice typing real code snippets in Java, Python, and JavaScript with real-time performance metrics.

## Tech Stack

- **React 19** + **Vite**
- **Tailwind CSS v4**
- **React Router v7**
- **JetBrains Mono** font

## Design Philosophy

- Strict terminal engineering theme â€” no cards, no shadows, no gradients
- Muted contrast color system ("Low-Glow Terminal")
- Edge-aligned grid layout with structural dividers
- Block cursor for terminal authenticity

## Getting Started

```bash
npm install
npm run dev
```

## Features (V1)

- Language selection (Java, Python, JavaScript)
- Configurable session duration (30s, 60s, 120s)
- Real-time character-by-character typing comparison
- Live CPM, accuracy, and error tracking
- Countdown timer with auto-completion
- Session result summary
